CSV Format Guide

What your file needs to look like

Required columns

supplier_number supplier_name invoice_number department invoice_date amount_excl

Optional columns

vat_rate vat amount_incl

vat_rate defaults to 15% if not provided. amount_incl is calculated automatically.

Date format

invoice_date must be YYYY-MM-DD — e.g. 2025-03-15

supplier_number,supplier_name,invoice_number,department,invoice_date,amount_excl,vat_rate
S009,OfficeCo,OC-22119,Operations,2025-10-28,2175.00,15
S011,PaperMart,PM-77891,Finance,2025-11-01,1020.00,15

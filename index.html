# Invoice Supplier Status Note

Invoice Records

Invoice # Supplier Department Excl. VAT VAT Incl. VAT Date Status
Loading…

CSV File Format

What your spreadsheet needs to include

Required columns

supplier_number supplier_name invoice_number department invoice_date amount_excl

Optional columns

vat_rate vat amount_incl

If vat_rate is omitted, 15% VAT (South Africa) is applied automatically. If amount_incl is omitted, it is calculated from amount_excl + vat.

Validation rules

  • ✓ invoice_date must be in YYYY-MM-DD format and cannot be a future date.
  • ✓ Duplicate invoices (same supplier + invoice number) are automatically skipped.
  • ✓ All amounts are rounded to 2 decimal places. VAT must match within ±R 0.01.

Example

supplier_number,supplier_name,invoice_number,department,invoice_date,amount_excl,vat_rate
S009,OfficeCo,OC-22119,Ops,2025-10-28,2175.00,15
S009,OfficeCo,OC-22120,Sales,2025-10-29,450.00,15
S011,PaperMart,PM-77891,Ops,2025-11-01,1020.00,15
